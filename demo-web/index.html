Machine Translation Applications to Historical Documents

Demo of some neural machine translation applications to historical documents.

Instructions:

  1. Select a task:
    1. Language modernization: rewrite a document's content using a modern version of its original language.
    2. Spelling normalization: make a document's orthography consistent according to modern standards.
  2. Read an original sentence.
  3. Click on the "Modernize"/"Normalize" button for obtaining the automatic modernization/normalization of the original sentence.
  4. If the modernization/normalization is not correct, you can make corrections:
    1. Use the mouse or the keyboard arrows to select the position where to introduce the correction.
    2. Type the correction.
    3. Type "/" for introducing an end-of-sentence symbol.
  5. The system will react to these interactions, providing alternative hypotheses.
  6. Once you are satisfied with the modernization/normalization, click in the "Validate" button and go to the next sentence.
  7. If Learn from sample is activated (in blue), the system will use this sentence pair for improving the model.
